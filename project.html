Überblick über Ziel, Architektur und Erweiterungsmöglichkeiten.

Ziel

LineIQ erfasst Produktions-, Fehler- und Nacharbeitszahlen, speichert Auftragsdaten strukturiert und stellt einen Fullscreen-Dashboard-Client bereit.

Komponenten

  • Flutter Desktop Dashboard
  • FastAPI Backend (REST)
  • Dienste: Scanner, GPIO, Systemstatus
  • Filesystem-Struktur (Year/Month/Day)

Architektur (High-Level)

[Flutter Dashboard]
      |
      |  REST / WebSocket (optional)
      v
[FastAPI Server] --- [Scanner Service]
      |
      +--- [GPIO / Hardware Service]
      |
      +--- [Filesystem Storage]
