NOT-AUS: Kamera-Schwenk + Signal → Raspberry(GPIO) → Code Overlay → Display

Ein Fenster, drei Stationen in einer breiten Szene. Beim Klick schwenkt die „Kamera“ automatisch von links nach rechts.

Bereit
Station 1 — Taster
Drücken → Signal startet
Das Signal läuft auf der Leitung los. Danach schwenkt die Kamera zum Raspberry, dort wird „Code“ ausgeführt, und anschließend geht’s weiter zum Display.
Kamera bleibt zunächst links. Signal läuft → dann Schwenk zum Raspberry.
Station 2 — Raspberry (GPIO)
Raspberry Pi (Symbolisch, als „Bild“) Idle
Backend-Handler (Dummy) wartet…
Hier: Signal trifft GPIO → Overlay zeigt Code-Ausführung → danach geht’s zum Display.
Station 3 — Display
Display (Dummy) Bereit

STATUS: OK

System läuft. Keine Eingriffe.

Zum Schluss: Signal erreicht Display → kurzer Flicker → Textwechsel.
